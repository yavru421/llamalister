LlamaLister Portable
====================

This is a portable version of LlamaLister that can run on any Windows computer with Python installed.

Requirements:
- Python 3.8 or higher
- Windows 10 or later

To run:
1. Double-click "LlamaLister_Launcher.bat"
2. Or run: python llamalister\llamalister.py

The application will save all data within its own directory structure.

Features:
- AI-powered eCommerce listing generation
- Vision analysis for product images
- Overlay assist for external site filling
- Comprehensive settings and configuration
- Memory service for learning and context

For more information, see the main LlamaMachinery repository.
